<p>Pass-compatible password manager that stores passwords as encrypted (OpenPGP) text files. A community continuation of the original pass: project (GPL-3.0-only).</p>

<p>This app can also act as an Android WebAuthn/passkey Credential Provider, allowing stored credentials to satisfy passkey requests from other apps.</p>

<p>
<strong>Features:</strong>
<ul>
<li>Clone an existing pass repository or start a new one</li>
<li>Create and organize password files</li>
<li>Sync with a remote Git repository</li>
<li>Decrypt and copy passwords</li>
<li>Automatically fill and save credentials in apps and supported browsers</li>
<li>Supports time-based one-time passwords (TOTP) for two-step authentication</li>
<li>Provides WebAuthn/passkey authentication to other apps</li>
</ul>
</p>
